🔑 CREDENCIAIS QUE VOCÊ VAI PRECISAR:
═══════════════════════════════════════════════════════════════════════════════

1. SUPABASE_URL
   └─ Obter em: Supabase → Settings → API → URL

2. SUPABASE_ANON_KEY (para Frontend)
   └─ Obter em: Supabase → Settings → API → Anon public

3. SUPABASE_SERVICE_ROLE_KEY (para Backend/Vercel)
   └─ Obter em: Supabase → Settings → API → Service Role secret

4. API_KEY (MeuDanfe)
   └─ Obter em: Seu dashboard MeuDanfe
